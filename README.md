# Horizon Logistics Dispatcher System

## 📌 Project Overview
The Horizon Logistics Dispatcher System is a real-time logistics management dashboard designed for internal fleet operations. It provides dispatch teams with tools to manage orders, assign routes, track drivers, and monitor delivery performance in real time.

This project is built for demonstration and internal evaluation purposes.

---

## ⚙️ Key Features

- 📦 Order Management (create, track, update deliveries)
- 🚚 Driver Assignment & Tracking
- 🗺️ Route Management System
- 📊 KPI Dashboard (on-time delivery, fleet utilization, fuel efficiency)
- 🔄 Real-time updates simulation
- 🧑‍💼 Role-based structure (Admin / Dispatcher / Driver - planned)

---

## 🧱 Tech Stack

- Frontend: HTML, CSS, JavaScript *(or Next.js if applicable)*
- UI Design: Custom dashboard system (Navy & Gold theme)
- Real-time Logic: JavaScript interval simulation (upgradeable to WebSockets)
- Backend (planned/optional): Firebase / Node.js API

---

## 🎨 Design System

- Primary Color: Navy (#0B1F3A)
- Accent Color: Gold (#C89B3C)
- UI Style: Modern, clean, data-driven dashboard
- Layout: Responsive grid-based system
